Luck-Application
================
为啥写？
===============================================================================
2013年西安电子科技大学通信工程学院毕业晚会需要有个校授的抽奖环节，需要这个软件

然后呢？
===============================================================================
本来是准备将照片作为滚动图片的，因为某些原因，最终只用1-45号的图片进行代替，之后姓名和学号在底部显示。点击开始后，可以进行抽奖。
实话说，从产品角度来看，这确实是一个失败的产品。因为从观众角度来看，滚动的图片好像是一个不定的，而软件只是按照顺序进行挨个显示，但是最终结果却是固定的。当时在晚会的时候，底下观众的反馈也可以看出来(当时真是伤心至极，嘘声惨不忍睹啊)。

有啥注意的呢？
===============================================================================
1.环境是1024*768的全屏，VS2010的编译环境。
2.出于隐私的考虑，把姓名数据马赛克了。
